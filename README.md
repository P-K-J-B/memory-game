# Memory Matching Game

Simple browser game for a Udacity course end-of-module project.



The aim of the game is to flip over cards on the game board and match up the corresponding icons. Match the eight card types with their respective counterparts and you win! 



You will receive a star rating based on the amount of total moves it takes you to beat the game so keep trying to get all three stars and become a memory master.



### Requirements

Game requires an internet browser to be played.

Confirmed fully functional on: Google Chrome, Firefox & Microsoft Edge.

This game is fully responsive and can be played on any device size.
